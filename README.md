## 🚀 Featured Projects

### 🏙️ Smart City Platform
> *Cloud-native, event-driven microservices platform for connected urban device management*

A distributed system built with Java & Spring Boot where independent microservices communicate asynchronously through Apache Kafka. Implements OAuth2 security via Keycloak, independent PostgreSQL databases per service, and domain event-driven projections.

**Stack:** `Java 21` `Spring Boot` `Apache Kafka` `Keycloak` `PostgreSQL` `Docker`

[![Repo](https://img.shields.io/badge/GitHub-Smart_City_Platform-181717?logo=github)](https://github.com/ndongo-ndiaye/smart-city-platform)

---

### 🧠 Memorix
> *AI-powered quiz generation platform with RAG pipeline and session tracking*

Memorix lets users upload documents (PDF, Markdown, plain text) and instantly generates personalized quizzes using a Spring AI RAG pipeline. Results and session history are persisted and displayed in a dashboard. Secured with Keycloak + OAuth2. Observability via Prometheus & Grafana.

**Stack:** `Java 17` `Spring Boot` `Spring AI` `Keycloak` `Angular` `PostgreSQL` `Prometheus` `Grafana`

[![Repo](https://img.shields.io/badge/GitHub-Memorix-181717?logo=github)](https://github.com/ndongo-ndiaye/memorix)
